# nodegen

A CLI tool for creating node app

## Installation

```sh
$ npm install -g @realvikas/nodegen
```

## Quick Start

Create the app:

```bash
$ nodegen init <dir>
```

## Generated Project Structure

```bash
> bin
    - www.js
> src
    - server.js
> config
    - keys.js
> app
    > {component}
        - {component}.controllers.js
        - {component}.routes.js
    - app.js
.env
.gitignore
LICENSE
README.md
package.json
```
